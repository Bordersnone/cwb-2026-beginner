# Borders:None Web Development - Beginner 2026

Welcome to the Borders:None web development teaching repository for 2026! This repository contains course materials, class documentation, and practice exercises for our hands-on web development curriculum.

## 📚 What's Inside

- **[Class Documentation](/classes)** - Lesson materials, quizzes, and practice exercises for each class

## 🎓 About This Course

This is a hands-on web development course that takes students from zero to building and publishing their first web pages. Students learn:

- HTML fundamentals and document structure
- CSS for styling and layout
- JavaScript basics
- Building and publishing real web pages

## 📖 How We Teach: Live Documentation, Not Slides

Instead of slide decks, this course teaches directly from the same documentation professional developers use every day. Bookmark these — you'll use them in every class:

- **[W3Schools](https://www.w3schools.com/)** - Beginner-friendly tutorials with interactive "Try It Yourself" examples
- **[MDN Web Docs](https://developer.mozilla.org/)** - The official, most complete web development reference
- **[W3C HTML Validator](https://validator.w3.org/)** - Paste in your HTML and it finds your mistakes
- **[HTML Reference](https://htmlreference.io/)** - Visual guide to all HTML elements with examples

Each class folder includes a `resources.md` with the specific pages for that class's topics.

## 📖 Getting Started

If you're a student in this course:

1. Browse the [class documentation](/classes) to find your current lesson
2. Keep the class's `resources.md` links open while you build — looking things up is a core developer skill
3. Do the homework! Web development is learned by building, not by watching
