# 🧭 Side-Quest: GitHub Setup

Do this if you don't have a GitHub account yet (or never made a repository). Budget: ~10 minutes. Then get back on the ladder.

## 1. Create your account

1. Go to **[github.com/signup](https://github.com/signup)**
2. Use an email you can open **right now** (you'll need to verify it)
3. **Pick your username carefully** — it becomes part of your website address later (`username.github.io`) and you'll show it to future employers. Your name or something close beats `xX_dragon_Xx`.
4. Complete the puzzle/verification, then check your email for an **8-digit code** and type it into the GitHub page

**Stuck?** Username taken → add a short suffix (`-dev`, a number). No code email → check spam, then "Resend".

## 2. Create your first repository

A **repository** ("repo") is a project folder that lives on GitHub.

1. Click the **+** in the top-right corner → **New repository**
2. Name: `my-favorite-things`
3. Keep it **Public**
4. Check ✅ **"Add a README file"**
5. Click **Create repository**

## 3. Upload your project (no git needed!)

1. In your new repo, click **Add file → Upload files**
2. Drag your project files in from your file manager — HTML files, `style.css`, and the `images` folder with its contents
3. Click the green **Commit changes** button
4. Click around your repo — can you see every file? Click an HTML file: GitHub shows you the code

> ⚠️ GitHub shows your files as *code*, not as a rendered website. Making it render as a real website is **Round 15**. Patience.

## 4. Report in

Tell your teacher your username so they can find `github.com/YOUR-USERNAME/my-favorite-things`. Done — back to the ladder! 🧗
