# 📖 Reference Links — HTML & CSS Review

We don't use slides in this course — we use the same documentation professional developers use every day. These links cover everything in today's class. Keep this page open while you build!

**How to use these two sites:**

- **[W3Schools](https://www.w3schools.com/)** — beginner-friendly tutorials with **"Try It Yourself"** buttons that let you edit code live in your browser. Start here.
- **[MDN Web Docs](https://developer.mozilla.org/)** — the official, complete reference. More detail, more precise. Go here when you want the full story.

## Handy Extras

- **[W3Schools Tryit Editor](https://www.w3schools.com/tryit/)** — a blank sandbox to experiment in, no setup needed
- **[W3C HTML Validator](https://validator.w3.org/)** — paste your HTML and it finds mistakes for you (like missing closing tags!)
- **[HTML Reference](https://htmlreference.io/)** — a visual cheat sheet of every HTML element

> 💡 **Habit to build:** when you can't remember how a tag works, don't wait to ask — look it up on one of these sites first. Reading documentation *is* a developer skill, and today is a great day to start practicing it.
